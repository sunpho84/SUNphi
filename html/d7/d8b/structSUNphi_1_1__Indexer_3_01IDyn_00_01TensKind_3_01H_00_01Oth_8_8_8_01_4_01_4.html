Detailed Description

template<int IDyn, class H, class... Oth>
struct SUNphi::_Indexer< IDyn, TensKind< H, Oth... > >

Indexer class to compute an index for a TensKind

Recursive implementation definining a nested indexer and calling it until no component is found. The check on types is done in the externally visible routine.
