Detailed Description

template<typename T>
class SUNphi::SingleInstance< T >

Single Instance enforcer class

To enforce a class to be instantiated at most once, inherit this class. Example:

class A :
public SingleInstance<A>
{
};
A a1; // ok
A a2; // error!
